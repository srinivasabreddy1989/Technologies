## Git

### What is Git?
Git is a distributed version control system designed to handle everything from small to very large projects with speed and efficiency. It allows multiple developers to work on the same codebase simultaneously, tracks changes over time, and provides tools for collaboration, branching, and merging code changes. Git was created by Linus Torvalds in 2005 and has become the standard version control system used in software development worldwide.

Key features of Git include:
- **Distributed**: Every developer has a full copy of the repository
- **Branching and Merging**: Easy creation and management of branches for feature development
- **Speed**: Fast operations even with large codebases
- **Data Integrity**: Uses SHA-1 hashes to ensure data integrity
- **Collaboration**: Supports remote repositories for team collaboration
